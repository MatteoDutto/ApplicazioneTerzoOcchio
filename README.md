# Applicazione Terzo Occhio

Applicazione che permette di visitare i luoghi di interesse storico-culturale di Cuneo (punti di interesse: <a href="http://www.ilterzocchio.it/"><b>il Terzo Occhio</b></a>). Questa visita viene fatta sottoforma di gioco, in cui bisogna cercare i punti su una mappa generata appositamente.
